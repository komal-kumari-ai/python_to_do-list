# Python To-Do List

A simple To-Do List application developed in Python as part of the Industrial Training Program by DecodeLabs.

## Features
- Add tasks
- View tasks
- Store multiple tasks using Python Lists
- Simple and beginner-friendly interface

## Concepts Used
- Lists
- append()
- for loops
- User Input

## Training Program
This project was completed as Project 1 during the Python Programming Industrial Training Program powered by DecodeLabs.
